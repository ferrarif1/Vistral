# Vistral Frontend Design Source Of Truth

This repository currently uses [`notion/DESIGN.md`](./notion/DESIGN.md) as the authoritative frontend design specification.

Use that document for:

- color roles
- typography scale
- spacing and radius rules
- card, button, input, badge, stepper, and shell treatments
- unified chat-first workspace styling

If any implementation or older design note conflicts with `notion/DESIGN.md`, `notion/DESIGN.md` wins.
